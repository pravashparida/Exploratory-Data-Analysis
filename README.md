# Exploratory Data Analysis (EDA)

## Overview
This repository contains a complete Exploratory Data Analysis workflow covering data cleaning, feature engineering, statistical analysis, and business insight generation using real-world datasets.

## Datasets
- Telecom Churn Dataset - CustomerChurn.csv
- Churn_Modelling.csv
- raw_sales.csv

## EDA Techniques Covered
- Missing Value Handling
- Feature Scaling & Encoding
- Outlier Detection & Treatment
- Feature Binning
- Univariate, Bivariate & Multivariate Analysis

## Case Study: Telecom Churn Analysis
Key insights include:
- Higher churn among senior citizens
- Increased churn for customers without partners
- Monthly contract users show the highest churn
- Payment method significantly impacts churn
- High monthly charges correlate strongly with churn

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Outcome
The project demonstrates the ability to transform raw data into actionable business insights through structured EDA.
